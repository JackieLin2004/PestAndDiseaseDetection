# Pest And Disease Detection

### 项目主要借助 `YOLOv5` 来实现对病虫害的实时检测，以达到代替人工检测的目的

本项目并未给出数据集，但是训练的模型均保存在 `logs` 文件夹中，也可以按照 `Quoted_README.md` 中的指示训练自己的模型

### 主要借助的参考代码 [>>> 传送门](https://github.com/bubbliiiing/yolov5-pytorch)
